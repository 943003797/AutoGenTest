# AutoGenTest 项目指南

## 🎯 项目概述
基于 Microsoft AutoGen 框架构建的多智能体协作实验项目

## ⚙️ 环境配置

### 项目初始化

#### 创建虚拟环境
```bash
python -m venv venv
```

❀ Windows 系统
  ✿ 激活
    ```bash
    .\venv\Scripts\activate
    ```bash
❀ Linux/macOS 系统
  ✿ 激活
    ```bash
    source venv/bin/activate
    ```bash
❀ 退出虚拟环境
    ```bash
  deactivate
```bash
❀ 安装依赖
```bash
  pip install -m requirements.txt
```bash
